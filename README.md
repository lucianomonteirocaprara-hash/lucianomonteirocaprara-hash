## Projeto em destaque

### PeritoLab

O PeritoLab é uma plataforma desenvolvida para auxiliar na preservação e análise de evidências digitais.

Principais funcionalidades:

- Geração de Hash (MD5, SHA-1, SHA-256 e SHA-512)
- Verificação de Integridade
- Metadata Inspector
- Geração de Cadeia de Custódia
- Relatórios Técnicos
